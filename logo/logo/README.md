# ByteArena - Ігровий Магазин

## Структура проєкту

```
ByteArena/
├── app.py                 # Головний файл Flask-додатку
├── __pycache__/          # Кеш Python-файлів
│   └── *.pyc            
│
├── backup/               # Резервні копії файлів
│   ├── about.css
│   ├── biblioteka.css
│   ├── ByteArena.css
│   └── login.css
│
├── css/                  # Стилі сайту
│   ├── biblioteka.css    # Стилі для сторінки бібліотеки
│   ├── ByteArena.css     # Головні стилі сайту
│   ├── GostOfTsusima.css # Стилі сторінки гри Ghost of Tsushima
│   ├── login.css         # Стилі форми входу
│   ├── news.css          # Стилі сторінки новин
│   └── Register.css      # Стилі форми реєстрації
│
├── html/                 # HTML-сторінки
│   ├── about.html        # Про нас
│   ├── biblioteka.html   # Бібліотека ігор
│   ├── ByteArena.html    # Головна сторінка
│   ├── GostOfTsusima.html# Сторінка гри
│   ├── login.html        # Форма входу
│   ├── news.html         # Новини
│   ├── portfolio.html    # Портфоліо/Кошик
│   └── Register.html     # Форма реєстрації
│
├── img/                  # Зображення та медіафайли
│   ├── games/           # Зображення ігор
│   ├── backgrounds/     # Фонові зображення
│   └── icons/          # Іконки та логотипи
│
├── js/                   # JavaScript файли
│   ├── login.js         # Логіка форми входу
│   ├── protect.js       # Захист сторінок
│   ├── reg-log.js       # Спільна логіка входу/реєстрації
│   └── register.js      # Логіка форми реєстрації
│
└── settings.json         # Налаштування проєкту
└── users.json           # База даних користувачів
```

## Опис компонентів

### Серверна частина
- `app.py` - Flask-сервер, обробляє запити, керує сесіями та авторизацією
- `users.json` - Зберігає дані користувачів
- `settings.json` - Конфігурація проєкту

### Клієнтська частина
- **CSS/**
  - Містить всі стилі для різних сторінок сайту
  - Кожен файл відповідає за свою сторінку
  
- **HTML/**
  - Всі HTML-сторінки сайту
  - Структуровані по функціоналу (авторизація, каталог, сторінки ігор)

- **JS/**
  - JavaScript для інтерактивності
  - Модулі авторизації та захисту

- **IMG/**
  - Медіаресурси сайту
  - Структуровані по категоріях

## Як запустити проєкт

1. Переконайтеся, що встановлено Python 3.x
2. Встановіть залежності:
   ```
   pip install flask
   ```
3. Запустіть сервер:
   ```
   python app.py
   ```
4. Відкрийте http://localhost:5000 у браузері

## Функціонал

- Реєстрація та авторизація користувачів
- Каталог ігор з детальними сторінками
- Система покупок та кошик
- Особистий кабінет користувача
- Новини та оновлення
